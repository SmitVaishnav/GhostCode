# Changelog

## [0.1.0] - 2026-02-28

### Added
- **One-click Hide/Reveal** buttons in editor toolbar
- **Pre-Send Risk Report** with domain exposure assessment (LOW/MEDIUM/HIGH)
- **4 privacy levels** for progressive anonymization
- **Ghost Map sidebar** with symbol browsing and hover previews
- **Multi-file hide** with shared ghost maps
- **Encrypted map files** for at-rest security
- **Explorer context menu** integration
- **Keyboard shortcuts**: `Cmd+Shift+G H` (hide), `Cmd+Shift+G P` (hide project)
- Python and C/C++ language support
- Interactive walkthrough for first-time users
