# Explore the Ghost Map

After hiding a file, the **Ghost Map** sidebar appears in the Explorer panel.

## What You'll See

Symbols are grouped by kind:

- **Variables** — `gv_001 → quarterly_sales`
- **Functions** — `gf_001 → calculate_revenue`
- **Types** — `gt_001 → RevenueReport`
- **Constants** — `gc_001 → TAX_RATE`
- **Strings** — `gs_001 → "api.internal.com"`
- **Macros** — `gm_001 → MAX_BUFFER`
- **Namespaces** — `gn_001 → analytics`

## Hover for Details

Hover over any token in the editor to see:
- The **original name**
- The **symbol kind**
- The **scope** where it was found

## Load a Different Map

Click **Ghost: Show Map** in the Command Palette to load any previously saved map file.
